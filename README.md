    # React Landing Page + Admin Dashboard

A simple React project with landing pages, authentication, and a dashboard layout using **React Router** and **Tailwind CSS**.

---

## Features

- Landing Pages:
  - Home
  - Features
  - Contact
- Authentication:
  - Custom SignIn only
  - Default admin credentials:
    ```
    Email: admin@gmail.com
    Password: admin123
    ```
- Dashboard:
  - Sidebar with links: Dashboard, Profile, Settings, Analytics
  - Logout redirects to Home
  - Pages styled with white content area
- Responsive for mobile, tablet, and desktop
